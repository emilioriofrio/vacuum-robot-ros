# Vacuum Robot (ROS Noetic y Gazebo)

## Requisitos
- Ubuntu 20.04
- ROS Noetic (desktop-full)

## 1) Instalar dependencias (una sola vez)
```bash
sudo apt update
sudo apt install -y ros-noetic-desktop-full git
sudo apt install -y ros-noetic-gazebo-ros ros-noetic-gazebo-ros-pkgs ros-noetic-gazebo-ros-control
sudo apt install -y ros-noetic-xacro ros-noetic-robot-state-publisher ros-noetic-joint-state-publisher
